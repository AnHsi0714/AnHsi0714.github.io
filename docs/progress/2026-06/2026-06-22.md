# 2026-06-22 ~ 2026-06-28

> 每月一個資料夾（`docs/progress/YYYY-MM/`），每週一個檔案，檔名為當週週一日期。每天工作完在當週檔案下面新增一個日期區塊。

## 2026-06-22（一）

- 新增 `src/types/content.ts`，統一定義 `Dream`、`Project`、`Book` 型別。
- 夢想頁（`Dreams.tsx`）改為讀取 `content/dreams.json`，依狀態（尚未開始／進行中／已達成）以 `Badge` 顯示。
- 專案頁（`Projects.tsx`）改為讀取 `content/projects.json`，以卡片呈現截圖、描述與 GitHub 連結。
- 讀書頁（`Books.tsx`）大幅擴充：
  - 新增書籍範例資料（之後改讀 `content/books/*.md`）。
  - 實作置中聚焦的書籍 carousel：捲動時自動偵測置中項目並放大／提亮、左右切換按鈕、自訂捲動進度條，支援鍵盤操作。
  - 新增篩選／排序面板：書名搜尋、分類多選、最低評分、起訖日期區間、最新／最久排序，並顯示已套用篩選數量。

## 2026-06-23（二）

- 讀書頁資料來源從寫死陣列改為 `content/books/*.md`：
  - 每本書一個 `.md` 檔，frontmatter 寫 `title`／`author`／`rating`／`finishedOn`／`categories`，正文即心得。
  - 新增 `src/lib/books.ts`：用 `import.meta.glob` 以 `?raw` 方式 eager 載入所有書籍檔案，自行解析 frontmatter（只支援扁平 key: value 與 `[a, b]` 陣列，非完整 YAML），依 `finishedOn` 由新到舊排序。
  - `Books.tsx` 移除原本寫死的範例資料，改為 `import { books } from "../../lib/books"`。
- Carousel 捲動進度條從「捲軸把手」改成「圓點導覽」：移除 `scrollBar` 狀態與寬度／位移計算，改成每本書一個可點擊的圓點，點擊直接 `centerItem` 跳轉，當前項目以較大實心點標示。

  - 夢想區拿掉原本的 `status`（todo/doing/done）欄位，回歸純粹「想做的事＋為什麼想做」的靜態清單。

## 2026-06-28（日）

- 新增可重複使用的 `ExpandableCard` 元件：包住 `Card`，點擊／按 Enter／Space 後用 `createPortal` 在全螢幕置中遮罩顯示完整內容（圖片固定顯示在卡片與展開內容最上方），按 Esc 或點背景關閉；開啟時鎖定 `body` scroll 並做基本 focus 管理。`ComponentsPreview` 新增對應示範。
- 整理 `theme.css` 色彩變數命名與配色：
  - danger 系列拆成 `--color-danger`（底色）與 `--color-danger-text`（文字色），取代原本 `--color-danger`／`--color-danger-bg` 混用的命名，`tokens.scss` 同步更新對應的 SCSS 變數。
  - 重新調整 light／dark 兩套主題的色階（primary 改用深青色系、danger／success／info 改用實色＋固定淺色文字），讓 `Alert`／`Badge`／`Button`／`Loading` 等元件的對比更清楚；同步修正 `Projects.tsx`、`ComponentsPreview.tsx` 內舊變數名稱的用法。
  - 順手修正 `Button.module.scss` 的 `.primary` 邊框色：原本寫成 `border-color: $color-border 2px`，但 `border-color` 不接受寬度值，是不合法的 CSS，整條宣告會被瀏覽器忽略；改成分開寫 `border-color` 與 `border-width`。
- `Loading` 怪獸新增隨機配色：用 `useMemo` 在掛載時隨機挑一組（怪獸色、角色）透過 CSS variable 套用，讓每次出現的怪獸顏色不同；角的形狀從圓角矩形疊出尖角改成 `clip-path` 三角形並調整角度，造型更自然。配色一開始嘗試隨機 HSL 色相，後來改成固定挑選 5 組事先搭配好的色票，避免隨機組合出來不好看。
